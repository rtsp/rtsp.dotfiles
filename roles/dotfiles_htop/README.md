# README - Ansible `rtsp.dotfiles.dotfiles_htop`

## Compatibility

- `htoprc` in this role support htop v1.0 or later

## Supported Distro

- Debian 7 (Wheezy)
- Debian 8 (Jessie)
- Debian 9 (Stretch)
- Debian 10 (Buster)
- Debian 11 (Bullseye)
- Ubuntu 16 (Xenial Xerus)
- Ubuntu 18 (Bionic Beaver)
- Ubuntu 20 (Focal Fossa)
